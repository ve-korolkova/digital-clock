# digital-clock
Цифровые часы. Часы показывают текущее время и дату с возможностью выбора часового пояса.
Для отображения времени используется Intl.DateTimeFormat и toLocaleTimeString — это нативный и кроссбраузерный способ, который корректно учитывает часовые пояса и локализацию.


## 🚀 Как запустить

1. Скачай или клонируй репозиторий.
2. Открой файл `index.html` в любом браузере (Chrome, Яндекс Браузер, Edge).
3. Готово — часы сразу начнут показывать время!

> 💡 Если хочешь запустить через локальный сервер (как делают разработчики):  
> `python -m http.server`  
> затем открой в браузере `http://localhost:8000/clock.html`.
